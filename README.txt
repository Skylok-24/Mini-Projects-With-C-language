# Mini and simple Projects with C Language

## Description
This is my Mini Projects in C 

## Requirements
- C/C++ extension in VS CODE
- gdb/gcc compiler

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/Skylok-24/Mini-Projects-With-C-language
    ```
2. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
